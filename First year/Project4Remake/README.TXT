Laboratory work 1.
According to the given formula of the member of the series with the number k, compose two programs:
a) a program for calculating the sum of the first n terms of a given series;
b) a program for calculating all terms of the series, modulo no less than a given number E.
Formula: k/((k+1)(k+2))

Лабораторная работа 1.
По заданной формуле члена ряда с номером k составить две программы:
а) программу вычисления суммы первых n членов заданного ряда;
б) программу вычисления всех членов ряда, по модулю не меньших заданного числа E.
Формула: k/((k+1)(k+2))
